# HomePage UI – HTML & CSS
- A simple and responsive homepage UI built using pure HTML & CSS.
This project focuses on clean design, beginner-friendly structure, and reusable styling—ideal for practicing frontend fundamentals.

## Features
- Responsive navigation bar
- Hero section with call-to-action
- Clean typography & spacing
- Mobile-friendly layout
- Organized HTML & CSS structure

## Tech Stack
- HTML5
- CSS3
